Zabbix NOC Alerter

Conexão

Só o endereço base (sem /zabbix.php). Você define; nada vem hardcoded.
Vazio = usa o cookie da sessão logada no navegador (sem token).

Monitoramento

A cada quantos segundos checa o Zabbix (mín 5s).
Só alerta deste nível pra cima.
Esconde problemas cujo nome OU host contém qualquer um destes textos (vírgula ou linha; ignora maiúsc/minúsc).
0 = mostrar todos. Use o mesmo valor do filtro do seu Zabbix (ex.: 14). Esconde os crônicos antigos.

Alertas

De quanto em quanto tempo re-toca enquanto houver problema não reconhecido.

Backup

Exporte/importe as configurações em JSON (o token de API não é exportado).
